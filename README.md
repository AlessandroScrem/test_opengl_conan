
# test_opengl_conan

C/C++ test demo using conan pakage manager https://conan.io/

## How to build this demo

Install Conan:

$ pip install conan

git clone https://github.com/AlessandroScrem/test_opengl_conan.git
cd test_opengl_conan
mkdir build && cd build
conan install ..
cmake ..
cmake --build
